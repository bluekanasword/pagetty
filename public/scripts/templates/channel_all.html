Welcome to Pagetty!

Since Pagetty differs from any other news reader/aggregator out there, you need to know some important concepts:

  • With Pagetty, you can turn practically any web page into a (RSS like) news feed.
  • You achieve this by defining CSS selector based aggregation rules for the web page that you want to track.
  • When you subscribe to a webpage and define the aggregation rules you have what we call, a channel.
  • Each channel will be automatically and periodically updated (like a RSS feed).
  • Rules are managed by users - everyone has a shared access to a rules for the sites they are subscribed to.
  • Rules are domain-specific - you don't need to configure rules for every single web page.
  • If someone has already created aggreagtion rules for the site you want to subscribe to then you do not need to deal with the rules at all.
  • Some CSS skills are probably required if you want to set up a channel where no rules have not been defined.
  • Don't turn back yet - there is also a wizard available, that automates the oh-so-difficult rule creation process :)

Awesome, right? Now, subscribe to some website to get started!
